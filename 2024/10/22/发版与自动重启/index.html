发版与自动重启

发表于2024-10-22|更新于2024-12-09
|阅读量:

平常我们直接 java -jar xxx.jar 启动项目,

创建run.sh 脚本如下

配置服务器重启 自动执行脚本

将脚本上传到服务器上后, 查看脚本权限, 用findshell的话, 直接右键 run.sh 选择文件权限, 勾选上全部的执行

看你的服务器支持什么, 我这里用的crontab

crontab -e 编辑crontab 文件

在文件里最后一行添加如下

@reboot /path/to/your/script.sh

:wq 保存并退出

crontab -l 检查文件

sudo systemctl reboot 重启服务器
